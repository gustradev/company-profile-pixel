# Company Profile — Pixel Solusindo
**CV Cipta Kreasitama**

Company Profile ini disusun sebagai dokumen resmi dan fleksibel untuk kebutuhan:
- Proposal proyek
- Tender (pemerintah / BUMN / swasta)
- Presentasi klien
- Arsip internal perusahaan

Dokumen menggunakan format **HTML + Tailwind CSS**, dengan fokus pada **print-ready (A4)** dan **konsistensi visual korporat**.

## 2. Tujuan Dokumen

Company Profile ini dirancang untuk:
- Menunjukkan kapabilitas teknis & operasional
- Menegaskan pendekatan kerja berbasis standar & SOP
- Memberikan gambaran struktur organisasi dan layanan
- Digunakan dalam format digital maupun cetak (PDF)

Pendekatan visual dan konten **tidak bersifat hard-selling**, melainkan **korporat, tenang, dan kredibel**.
